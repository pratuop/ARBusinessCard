# AR Business Card Generator

## 📌 Overview

The AR Business Card Generator is a web application that allows users to create, customize, and download professional business cards with ease. The application provides a dynamic interface where users can enter their details and preview the card in real time.

---

## 🚀 Features

* Dynamic business card customization
* Real-time preview functionality
* Download business cards as image or PDF
* SQL Server database integration
* Responsive and intuitive UI

---

## 🛠️ Technologies Used

* ASP.NET Core (C#)
* SQL Server
* HTML
* CSS
* JavaScript
* Bootstrap

---

## 📂 Project Structure

* **Controllers** – Handles application logic
* **Models** – Defines data structure
* **Views** – UI components (Razor/HTML)
* **wwwroot** – Static assets (CSS, JS, Images)

---

## ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/ARBusinessCard.git
```

### 2. Open in Visual Studio

Load the solution file in Visual Studio.

---

### 3. Configure Database

Update the connection string in `appsettings.json`:

```json
"DefaultConnection": "Server=YOUR_SERVER;Database=ARBusinessCard;Integrated Security=True;TrustServerCertificate=True;"
```

---

### 4. Apply Migrations (if applicable)

```bash
Update-Database
```

---

### 5. Run the Application

```bash
dotnet run
```
